### 指示
電子カルテシステムの過去カルテ領域のスクリーンショットです。$image_desc
以下のエントリが前回の画面で途中から切れている可能性があります。画像とOCRテキストを参照し、その続きが表示されているか確認し、続きのテキストのみを返してください。

### 確認対象エントリ
日付: $date
現在取得済みの末尾テキスト:
```
$tail_display
```

### タスク
画像・OCRテキストの中に上記エントリの続き（末尾テキストの直後から続くテキスト）が含まれている場合、その続きのテキストのみを返してください。
続きがない、または末尾テキストが既に完結している場合は continuation に空文字列を返してください。

### 重要な制約
- 続きとして返すテキストは末尾テキストの直後から始まる部分のみです。末尾テキスト自体は含めないでください。
- 新しい日付のエントリのテキストは含めないでください。
- 日付スタンプと本文の対応は画像の空間的な位置関係で判断してください。
- 画像・OCR認識テキストに存在しないテキストを生成しないでください。

### OCR認識結果（参考）
```
$ocr_text
```

### 出力フォーマット（JSONのみ）
{"continuation": "続きのテキスト（なければ空文字列）"}
